Contribute and Support

Support the Project

Valet is an open-source project that relies on community support. Here are ways you can contribute:

Financial Support

Help fund the development and maintenance of Valet:

  • Donate Bitcoin to our development fund
  • Support through Lightning Network payments
  • Become a monthly contributor

Technical Contributions

Help improve Valet through code contributions:

  • Report bugs and issues on GitHub
  • Submit pull requests for improvements
  • Help with code review and testing
  • Contribute to documentation

Community Support

Help grow and support the Valet community:

  • Help other users in our support channels
  • Create educational content
  • Translate the app to your language
  • Share your experience and feedback

Development Roadmap

We're always working on improving Valet. Here are some upcoming features we're developing:

  • Enhanced privacy features
  • Additional Lightning Network capabilities
  • Improved user interface and experience
  • More language support
  • Advanced security features

Note: The roadmap is subject to change based on community feedback and development priorities.
